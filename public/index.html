Basic usage

  • /api/centers
    • GET
      • returns all centers
    • POST
      • creates a new center with JSON as follows
        • name: STRING (60)
        • maxLanes: NUMBER (existing Center ID)
        • address: STRING (255)
  • /api/members
    • GET
      • returns all members for all centers
    • POST
      • creates a new member with JSON as follows
        • name: STRING (100)
        • centerId: NUMBER (existing Center ID)
        • skillLevel: NUMBER 0-4
    • PATCH
      • URL: /api/members/MEMBER_ID
      • Edits an existing member with JSON containing
        • name: STRING (Only supports name changes)
    • DELETE
      • URL: /api/members/MEMBER_ID
      • Deletes an existing member
